📋 General Requirements for All Roles
ⓘ What We Look For

Regardless of the specific role, all candidates at Hot Beans Web are expected to bring a positive attitude, a willingness to learn and the ability to work well in a team. Below are the core qualities and requirements that apply across all positions.

💻 Technical Foundation

All applicants must have a strong understanding of core web technologies including HTML, CSS and JavaScript. Knowledge of responsive design principles and accessibility standards is essential.

🎓 Education & Experience

A relevant degree, bootcamp certificate or demonstrable self-taught experience is required. We value practical skills and a strong portfolio above formal qualifications.

💡 Problem Solving Skills

You should be able to think logically, break down complex problems and find efficient, elegant solutions. We look for developers who enjoy the challenge of debugging and continuous improvement.

🤝 Communication

Strong verbal and written communication skills are essential. You will be working with clients, designers and developers daily, so the ability to explain technical concepts clearly is very important.

📈 Continuous Learning

The web industry moves fast. We expect all team members to stay curious, keep learning and actively engage with new tools, frameworks and best practices as they emerge.

🕐 Reliability & Punctuality

We trust our team to deliver on time and to the standard agreed. Good time management, meeting deadlines and proactively communicating any blockers are all essential qualities.

📄 Coding Languages & Technologies

HTML5

The foundation of all web pages. A solid understanding of semantic HTML is required for all roles.

CSS3

Styling, animations, flexbox and grid. All candidates should be comfortable writing clean, maintainable CSS.

JavaScript

Core programming logic, DOM manipulation and event handling. ES6+ knowledge is expected.

Bootstrap

Familiarity with a CSS framework like Bootstrap or Tailwind is desirable for most roles.

React / Vue

Experience with a modern JS framework is required for mid-level and senior positions.

Git & Version Control

All developers must be comfortable using Git for version control and collaborating on shared codebases.
